|type|what to use|example|
|---|---|---|
|Array|`.length`|`int[] arr = new int[5];` → `arr.length`|
|String|`.length()`|`"hello".length()`|
|List/ArrayList|`.size()`|`list.size()`|
|HashMap|`.size()`|`map.size()`|

**simple rule to remember:**

- arrays → `.length` (no brackets, it's a property)
- strings → `.length()` (brackets, it's a method)
- everything else (List, Map, Set) → `.size()`
