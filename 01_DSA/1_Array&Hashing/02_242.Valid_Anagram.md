## ASCII Codes

**Full form: American Standard Code for Information Interchange**

**Total: 128 characters** (0-127)

Includes:

- Lowercase letters: a-z (97-122)
- Uppercase letters: A-Z (65-90)
- Numbers: 0-9 (48-57)
- Symbols: @, #, $, !, ?, etc.
- Control characters: space, tab, newline, etc.

**Problem:** Only 128 characters. Not enough for the world.
## Unicode Codes

**Total: Over 1.1 million characters** (and growing)

Includes:

- All ASCII (0-127)
- All world languages
- Emojis
- Math symbols
- Musical notes
- Everything

**Range:** 0 to 1,114,111 (theoretically)

**Common ranges:**

- 0-127: ASCII (English)
- 128-255: Extended (accented letters, some symbols)
- 256+: Everything else (Chinese, Arabic, emojis, etc.)

**In practice:** Most of those 1.1 million aren't assigned yet. New characters added every year.

'a' has ASCII value 97
'b' has ASCII value 98
'z' has ASCII value 122

'a' - 'a' = 97 - 97 = 0    ← Store 'a' at index 0
'b' - 'a' = 98 - 97 = 1    ← Store 'b' at index 1
'z' - 'a' = 122 - 97 = 25  ← Store 'z' at index 25
