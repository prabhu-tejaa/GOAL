this is Kadane's algorithm approach

Kadane's algorithm is just the idea of:
- Keep a running sum
- If it goes negative, reset to current element
- Track the max at each step
It's a classic greedy algorithm named after Joseph Kadane.

```
    class Solution {

        public int maxSubArray(int[] nums) {

            int numsLen = nums.length;

  

            if(numsLen == 1){

                return nums[0];

            }

  

            int maxSubarray = Integer.MIN_VALUE;

            int runningSum = 0;

  

            for(int num : nums){

                runningSum += num;

                maxSubarray = Math.max(runningSum, maxSubarray);

  

                if(runningSum < 0){

                    runningSum = 0;

                }

  

            }

        return maxSubarray;

        }

    }
```
