## 1. Screening Round (Recruiter)

- **Focus:** Eligibility and Logistics.
    
- **Prep:** Prepare your 30-second "elevator pitch," salary expectations, and work authorization details.
    
- **Key Question:** "Why are you looking to leave your current role/school for Indeed?"
    

## 2. Technical Phone Screen (HackerRank)

- **Focus:** Data Structures (DSA).
    
- **Prep:** Master **Stacks, HashMaps, and Arrays**. Specifically, practice the "Min Stack" implementation and its time/space trade-offs.
    
- **Key Skill:** Explaining why $O(1)$ time is better than $O(n)$ for retrieval.
    

## 3. Resume Deep Dive Lite

- **Focus:** System Ownership and Metrics.
    
- **Prep:** Pick one complex project. Be ready to explain the **End-to-End flow** (from UI to Database).
    
- **Key Details:** Have numbers ready (e.g., "handled 500 requests per second" or "reduced latency by 20%").
    

## 4. Programming Exercise with AI (ProgEx)

- **Focus:** Functional Implementation with AI.
    
- **Prep:** Practice using **GitHub Copilot** to write a "Rule Engine" (logic that filters data based on conditions).
    
- **Key Skill:** "Context Engineering"—how you prompt the AI to help you optimize or debug a large coding task quickly.
    

## 5. Code Review (Java & Angular)

- **Focus:** Quality and Communication.
    
- **Prep:** Study React "Hooks" (useEffect/useState) and Python "Best Practices" (PEP 8). Learn to spot **Production Risks** like missing error handling or API security flaws.
    
- **Key Question:** "How do you handle a teammate who disagrees with your review?"
    

## 6. Whiteboard (Strings)

- **Focus:** Algorithm Logic.
    
- **Prep:** Practice **Sliding Window** and **Two-Pointer** techniques. Focus on string manipulation (finding keywords within a text and returning a "context snippet").
    
- **Key Skill:** Handling edge cases like case-insensitivity or empty strings.
    

## 7. Closing Round

- **Focus:** Cultural Alignment.
    
- **Prep:** Research Indeed’s mission: "We help people get jobs." Have 3 questions ready to ask the interviewer about the team culture or engineering challenges.
    
- **Key Question:** "Why Indeed specifically over other tech giants?"
